# Autoevaluación Entrega 5 Práctica - Grupo15

## Resumen del trabajo realizado.
En esta quinta entrega nuestro objetivo ha sido transformar nuestro dataset y transformarlo a un grafo RDF, utilizando RML como lenguaje de mapeo.

## Acciones detalladas
1.- Hemos empezado editando nuestro dataset con la herramienta de OpenRefine, creando nuevas columnas, que conectaban los nombres de las bibliotecas, los ditritos y barrios con la base de datos Wikibase, utilizando las uris que nos proporcionaba,.

2.- Luego hemos descargado este nuevo csv, con su json correspondientes.

3.- Usando la herramienta Matey, hemos creado el rml, a traves de un YARRRML creado correspondiente a nuestro csv.

4.- Después de crear el RML y descargarlo, hemos creado en notas un comando config.ini para poder usar el comando morph, que nos generaria el archivo RDF.

5.- Al corregir, y ver que todo funcionaba bien, hemos hecho las queries, y comprobado que funcionaban.

## Conclusiones:
Hemos conseguido actualizar nuestro Dataset para poder usar las uris, y asi nos permite hacer más consultas SPARQL
